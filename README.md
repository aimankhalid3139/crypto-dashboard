# 📊 Cryptocurrency Data Visualization Dashboard

An interactive, professional-grade Exploratory Data Analysis (EDA) dashboard built using **Streamlit**, **Pandas**, **Matplotlib**, and **Seaborn**. This project implements dynamic cross-filtering linked across 10 mandatory visualization chart types.

---

## 🛠️ Installation & Setup

Follow these steps to set up the environment and run the dashboard locally on your system:

### 1. Project Directory Structure
Ensure your local project folder is organized exactly as follows:
```text
/dashboard_project/
├── data/
│   └── cryptodata.csv          # Original shared dataset (unmodified)
├── notebooks/
│   └── analysis.ipynb          # Exploratory Data Analysis notebook
├── app.py                      # Main UI and dashboard setup
├── charts.py                   # 10 mandatory chart functions
├── filters.py                  # Sidebar dynamic filtering logic
├── requirements.txt            # Required library dependencies
└── README.md                   # Project documentation
